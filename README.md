## PikPak이 뭔데? — 일반 클라우드랑 뭐가 달라?

이게 뭐가 좋냐고? 몇 가지 장점이 확실해.

- **속도**: 엔터프라이즈급 네트워크 인프라라서, 80% 이상의 파일이 몇 초 안에 클라우드로 들어와
- **내 컴퓨터 안 켜도 됨**: 서버에서 다운로드하는 거니까 노트북 꺼도 OK
- **바로 스트리밍**: 저장된 영상을 굳이 내려받지 않고 클라우드에서 바로 4K로 재생 가능

무료 계정으로도 **6GB** 저장공간을 주는데, 이 정도면 일단 써보기엔 충분하다.
